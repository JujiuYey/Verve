# 07 · 指标与埋点

## 指标体系

**北极星**:每周完成 ≥1 个「验证通过的小目标」的活跃用户数。

### 激活漏斗

```text
注册 → 创建目标 → 路线生成成功 → 开始第一节 → 完成验证 → 验证通过(aha)
```
每一步转化率都要看,**重点盯**:
- 「路线生成成功率」(影响第一印象);
- 「首次验证通过率」(aha moment 是否达成)。

### 留存

次日 / 7 日 / 30 日回访;周学习活跃天数。

### 学习成效

验证通过率(`exercises.verdict = pass` 占比)、掌握层级提升速度、薄弱点消除率。

## 关键埋点事件(草案)

| 事件 | 触发时机 | 关键属性 |
|---|---|---|
| `sign_up` / `login` | 注册 / 登录 | |
| `goal_created` | 创建学习目标 | source=text |
| `path_generated` | 路线生成完成 | success、耗时、阶段数 |
| `session_started` | 开始一节 | objective_id |
| `exercise_submitted` | 提交练习 | type |
| `exercise_verdict` | 判定完成 | verdict、mastery_after |
| `session_completed` | 完成一节 | 时长、消息数 |
| `continue_clicked` | 点击继续上次 | |
