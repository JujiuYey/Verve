# 产品需求文档(PRD)

> 多 Agent 学习平台 · 监督式 AI 私教 —— 标准 PRD

本目录是**标准产品需求文档**,关注"做什么 / 给谁 / 为什么 / 长什么样"。技术方案(领域模型、Agent 实现、表结构 DDL)在 [`../product/`](../product/) 以及未来的 `../architecture/`。

## 两套文档分工

| | docs/prd/(本目录) | docs/product/ |
|---|---|---|
| 关注 | 需求、用户、交互、指标 | 产品形态 + 技术方案 |
| 典型内容 | 用户故事、线框图、成功指标、非功能需求 | 领域模型、Agent 设计、表结构 DDL |
| 面向 | 产品 / 设计 / 业务对齐 | 工程实现 |
